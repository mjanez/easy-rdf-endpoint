What is an RDF Endpoint?

An RDF endpoint is a web service that provides access to structured data in Resource Description Framework (RDF) format. This endpoint provides access to data catalogs that follow the W3C Data Catalog Vocabulary (DCAT) standards.


SPARQL and YASGUI

Our endpoint includes YASGUI (Yet Another SPARQL GUI), a user-friendly interface for querying RDF data. YASGUI offers:

  • Syntax highlighting for SPARQL queries
  • Auto-completion of SPARQL keywords and prefixes
  • Multiple output formats (JSON, XML, CSV)
  • Query result visualization options
  • Share and permalink queries

Available Features

  • RDF Catalog: Access the complete data catalog in RDF/XML format
  • SPARQL Editor: Interactive YASGUI interface for writing and testing queries
  • SPARQL Endpoint: Machine-readable endpoint for programmatic access
  • SHACL Validator: RDF validator using SHACL shapes
  • DCAT specifications compliance
  • Support for complex SPARQL 1.1 queries
